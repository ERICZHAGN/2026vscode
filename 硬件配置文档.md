# STM32U575 项目硬件配置文档

## 项目概况

- **MCU:** STM32U575RIT6 (LQFP64)
- **系统时钟:** 160 MHz (PLL 源 = MSI, PLLM=3, PLLN=10, PLLR=1)
- **RTOS:** FreeRTOS (CMSIS-RTOS V2, Heap4)
- **工具链:** CMake / GCC
- **调试串口:** USART1 (PA9-TX, PA10-RX, 波特率由 CubeMX 配置)

---

## 外设总览

| 外设 | 描述 | 通信方式 |
|------|------|----------|
| LED | 板载指示灯，PC13 低电平点亮 | GPIO 推挽输出 |
| OLED 显示屏 | 0.96寸 128x64 OLED, SSD1306 | I2C1 (地址 0x78) |
| MPU6050 | 6 轴加速度计/陀螺仪 | I2C1 (地址 0x68/0x69) |
| SG90 舵机 | 180° 模拟舵机 | TIM2_CH2 PWM |
| 光敏传感器 | 光敏电阻模块 | ADC1_CH8 模拟输入 |
| DHT11 | 温湿度传感器 | 单总线 (PA12) |
| W25Q64 | 8MB SPI Flash | SPI1 (PA4-CS) |
| 物理按键 x2 | 通用输入按键 (PC8、PC9) | GPIO 输入/中断 |

## 通信协议汇总

| 协议 | 总线 | 外设 |
|------|------|------|
| I2C1 | PB6 (SCL), PB7 (SDA) | OLED (0x78), MPU6050 (0x68/0x69) |
| SPI1 | PA5~PA7, PA4(CS) | W25Q64 |
| USART1 | PA9 (TX), PA10 (RX) | 调试串口 (printf) |
| ADC1 | PA3 (CH8) | 光敏传感器 |
| TIM2_CH2 PWM | PA1 | SG90 舵机 |
| 单总线 (GPIO) | PA12 | DHT11 |

**I2C1 总线共享说明：** OLED 和 MPU6050 共用 I2C1 总线，为避免冲突，FreeRTOS 中使用 `i2cMutexHandle` 互斥锁保护 I2C 访问。

---

## FreeRTOS 任务

| 任务名 | 入口函数 | 优先级 | 栈大小 | 功能 | 周期 |
|--------|----------|--------|--------|------|------|
| defaultTask | `StartDefaultTask` | Normal | 512B | 读取光敏传感器并打印 | 300ms |
| ledTask | `ledtoggle_task` | Low | 512B | LED 闪烁 | 500ms |
| mpu6050Task | `mpu6050_task` | Normal | 512B | 读取 MPU6050 数据 → 发送队列 | 100ms |
| oledTask | `oled_task` | Normal | 512B | 接收队列 → 显示在 OLED | 消息触发 |
| sg90Task | `sg90_task` | Low | 512B | 执行 0°→90°→180° 角度序列 | 一次 (含延时) |
| w25q64Task | `w25q64_task` | Normal | 1024B | SPI Flash 读写验证测试 | 一次 |
| dht11Task | `dht11_task` | Low | 2048B | 读取 DHT11 → 显示在 OLED (目前注释) | 2s |

**任务间通信：**
- `myQueue01Handle` — 消息队列，`mpu6050_task` 发送 `Mpu6050_Data_t` 结构体，`oled_task` 接收并显示

**同步原语：**
- `i2cMutexHandle` — 保护 I2C1 总线访问
- `spiMutexHandle` — 保护 SPI1 总线访问
- `spiDmaSemaphoreHandle` — SPI DMA 传输完成信号量

---

## 数据流

```
MPU6050 (I2C)
    → mpu6050_task (每100ms读取)
    → myQueue01Handle (消息队列)
    → oled_task (接收数据显示)

光敏传感器 (ADC)
    → StartDefaultTask (每300ms读取)
    → printf (串口打印)

SG90 舵机 (PWM)
    → sg90_task (角度序列 0°→90°→180°)

W25Q64 (SPI)
    → w25q64_task (写入/读取/擦除测试)

DHT11 (单总线)
    → dht11_task (每2s读取，目前已注释)
    → 待显示到 OLED
```
